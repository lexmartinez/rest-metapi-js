rest-metapi-js

Dynamic REST API: Just create a MongoDB collection and it's all.. you got a RESTful Web Service to manage that collection.

  • Avoid any back-end implementation.
  • Feel free to focus on improving your front-end.
  • No additional commands needed.

Based on rest-metapi (Java implementation)


Service Endpoints

Every RESTful Webservice includes this endpoints:

  • findAll
    GET: /api/{collection}
  • getById
    GET: /api/{collection}/{id}
  • insert
    POST: /api/{collection}
  • query
    POST: /api/{collection}/query
    //REQUIRES JSON Body request with search prototype
  • update
    PUT: /api/{collection}/{id}
  • delete
    DELETE: /api/{collection}/{id}
